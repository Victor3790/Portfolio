Smart English

Página para la escuela de inglés Smart English. Web estática de una sola página (One page), 100% responsiva y pensada para posicionarse en búsquedas locales. Para lograr un buen posicionamiento en buscadores se optimizó el texto para palabras clave definidas y se trabajó para obtener velocidades de carga óptimas (90% o más en pagespeed insights) por medio de la carga asíncrona de recursos. Se logró posicionar el sitio en la primera página en poco tiempo.

Rock Legends México

Proyecto en Wordpress para el blog de la web Rock Legends México. Para esta página se modificó el tema principal por medio de un tema hijo; el resultado fue una página ligera y responsiva lo cual facilitó el posicionamiento en buscadores; además se reforzó la seguridad del sitio cambiando la URL de la página de admin y añadiendo una segunda capa de seguridad poniendo una contraseña al folder de almacenamiento directamente en el servidor; también se instaló un plugin de límite de intentos de acceso.

Mezcal el Joven Viejo

Web para la marca de mezcal artesanal El Joven Viejo. Este sitio se hizo en Wordpress ya que se tiene la idea de escalarlo en un futuro para tener un blog y una tienda en línea. Para hacerlo usé un tema hijo y se optimizó por medio de plugins y con código para obtener 90% o más de calificación en Google Page Speed. La página se adapta a cualquier tamaño de pantalla.

Colectivo de poetas Verso Libre

Proyecto en Wordpress para el colectivo de poetas Verso Libre. El requerimiento principal era que cada miembro del colectivo tuviera acceso al sitio para publicar sus escritos. Se dieron de alta perfiles de autor. También se incluyó una revista digital editada por el grupo en una barra lateral del sitio. La página se adapta a cualquier tamaño de pantalla.
