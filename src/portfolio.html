Smart English

Página para la escuela de inglés Smart English. Uno de los requerimientos era que fuera una One page con la ubicación de la escuela. Al implementar la ubicación con google maps la velocidad de carga bajó considerablemente por lo que decidimos usar la API de Javascript de google maps para cargar el mapa de manera asíncrona. Logramos posicionar esta página para búsquedas locales optimizando el texto para palabras clave específicas y el plan a mediano plazo es desarrollar una estrategia de SEM.
